# 🍄 超级马里奥小游戏

经典的超级马里奥复刻版网页游戏！

## 🎮 游戏特色

- ✨ 流畅的跳跃和移动
- 🪙 金币收集系统
- 👾 敌人挑战
- 🏆 分数统计
- 🎨 可爱的像素风格
- 📱 响应式设计

## 🎯 游戏规则

### 操作按键

| 按键 | 功能 |
|------|------|
| ← / A | 向左移动 |
| → / D | 向右移动 |
| 空格键 | 跳跃 |
| R | 重新开始 |

### 游戏玩法

1. **收集金币** - 每个金币 +100 分
2. **击败敌人** - 从上方跳跃踩踏敌人，每个 +200 分
3. **保持存活** - 避免接触敌人，不要掉出地图
4. **探索平台** - 利用多跳到达更多金币位置

### 游戏技巧

- 💡 跳跃时可以控制方向
- 💡 从高处落下会对敌人造成更大伤害
- 💡 掌握跳跃时机很重要
- 💡 收集所有金币获得最高分
